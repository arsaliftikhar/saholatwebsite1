About App

Sahoolat is an Uber of Helping Hands. Each user in the network can sign up to be a provider and earn cash. A user creates a request for a product, service or a ride share. Providers providing that service bids on the request. Once the bid is accepted, the provider delivers the product or services and receive cash on delivery.

Sahoolat will revolutionize the way you think about unwanted tasks. If you need something done on-the-fly, no problem Sahoolat is there to make sure you get everything you need done.
